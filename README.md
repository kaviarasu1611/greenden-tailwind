# GreenDen 🌿

GreenDen is a simple and responsive plant shop website built using HTML and CSS. This project showcases a modern landing page design for an online plant store, featuring product sections, customer reviews, and a clean user interface.

## 🚀 Features

- Responsive navigation bar
- Attractive landing page
- Featured plant collections
- Customer reviews section
- Search functionality
- Clean and modern UI design
- Fully responsive layout

## 🛠️ Technologies Used

- HTML5
- CSS3
